# [DeployPRO](https://deploypro.dev/) `Flask Sample` 

Minimal `Flask Sample` provided by [DeployPRO](https://deploypro.dev/) service for testing.

<br />

## Manual Build

> Install modules via `VENV`  

```bash
$ pip install flask
```

> Start the app

```bash
$ flask run
```

At this point, the app runs at `http://127.0.0.1:5000/`.
